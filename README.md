# Aplikasi Monitoring Diet (EL-DIET)
Aplikasi mobile untuk membantu pengguna mengelola pola makan, konsumsi air, olahraga, dan progres diet harian secara terstruktur. Dikembangkan sebagai respon meningkatnya obesitas dan pola makan tidak sehat, EL-Diet berfokus pada kepatuhan target harian melalui fitur tracking, visualisasi progres, artikel edukasi, dan chatbot interaktif.
